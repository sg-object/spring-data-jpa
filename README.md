## spring-data-jpa

## 주요 기능
* Spring Data Jpa를 활용한 API

## Version
* Spring Boot : 2.3.0.RELEASE
* Swagger : 2.9.2

## H2 Database
* 간편한 테스트를 위해 H2 Database를 사용
* 초기 데이터 /src/main/resources/data.sql
* Console 접속 : http://localhost:8080/console
* Password는 설정되어 있지 않기 때문에 바로 접속 가능
